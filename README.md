# 基于 STM32 的直流电压电流功率监控器 (INA226 + SPI OLED)

这是一个基于 **STM32F103C8T6** 开发的直流系统参数监测项目。该设备利用 **INA226** 高精度功率监控芯片实时采集总线电压、电流和功率，并通过 **SPI OLED** 屏幕进行可视化显示。

## 🚀 项目特性

* **高精度测量**：基于 INA226 芯片，支持高达 1.25mV 的电压分辨率和 1mA 的电流分辨率。
* **高速显示刷新**：采用硬件/软件 SPI 驱动 OLED 屏幕，提供流畅的数据更新体验。
* **实时数据解析**：自动计算系统功耗（W）并监测分流电阻电压（uV）。
* **双模式支持**：代码包含真实传感器模式与模拟测试模式，方便在无硬件环境下调试 UI。

## 🔌 硬件接线 (Pinout)

### 1. SPI OLED 屏幕 (显示)
| OLED 引脚 | STM32 引脚 | 功能说明 |
| :--- | :--- | :--- |
| **CS** | **PB12** | 片选 (SPI2_NSS) |
| **SCK** | **PB13** | 时钟 (SPI2_SCK) |
| **RES** | **PB14** | 复位 (Reset) |
| **SDA** | **PB15** | 数据 (SPI2_MOSI) |
| **DC** | **PA9** | 数据/命令选择 |

### 2. INA226 传感器 (测量)
| INA226 引脚 | STM32 引脚 | 功能说明 |
| :--- | :--- | :--- |
| **SCL** | **PB6** | I2C 时钟线 |
| **SDA** | **PB7** | I2C 数据线 |

## 📂 项目结构

项目基于标准的 Keil MDK 工程架构开发：

* **Start**: 包含 STM32 启动文件及核心内核文件。
* **Library**: 标准外设驱动库 (FWLib)。
* **ADD**: 自定义驱动模块。
    * `INA226/`: 传感器初始化、寄存器操作及物理量转换逻辑。
    * `IIC/`: 模拟 I2C 通信时序实现。
    * `OLED/`: SPI 屏幕驱动及字符显示函数。
* **User**: 包含 `main.c` 主逻辑及中断处理程序。
